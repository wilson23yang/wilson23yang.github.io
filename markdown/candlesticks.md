# Candlesticks


实现K线库的基本功能与各项指标线：
    
 - 分时线、分钟线、5分钟线、15分钟线、30分钟线、1小时、日线、周线、月线
 - K线图缩放、历史节点指标查询、实时指标动态跟踪
 - MA、BOLL、RSI、KDJ、MACD、WR指标
 - 样式配置
 - 指标动态变化


```
CandlesticksWidget(
    durationMs: durationMs,
    dataStream: snapshot.data,
    candlesticksStyle:style,
)
```
